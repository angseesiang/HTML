# HTML Learning Projects Collection

## Overview

This repository contains **28 beginner-friendly HTML projects** created as part of my web development learning journey. Each project focuses on a different part of HTML, starting from simple headings and paragraphs, then gradually moving into forms, tables, media elements, semantic layout, accessibility, and structured webpages.

The purpose of this collection is to document my progress as I build a strong foundation in HTML. These projects may look small individually, but together they represent consistent practice, curiosity, and the first important steps toward becoming a better web developer.

## Skills Practiced

Through these projects, I practiced the following HTML skills:

- Creating basic HTML page structure
- Using headings and paragraphs to organize content
- Adding links and images
- Creating ordered, unordered, and definition lists
- Building semantic page layouts
- Creating forms with labels, inputs, checkboxes, radio buttons, dropdowns, and text areas
- Building accessible form and media controls
- Creating tables with headers, captions, body rows, footers, and column spans
- Embedding audio, video, subtitles, SVG graphics, and iframe videos
- Using accessibility attributes such as `aria-label`, `aria-labelledby`, `aria-describedby`, and `aria-hidden`
- Debugging and improving HTML structure

## What I Learned

These projects helped me understand that HTML is more than just tags. HTML gives structure and meaning to web content. A good webpage starts with clear organization, readable content, and thoughtful use of elements.

I learned that small details matter. A missing closing tag, incorrect attribute, or unclear label can affect how a page works. I also learned that accessibility is important because websites should be usable by as many people as possible.

Most importantly, I learned that consistent practice creates progress. Each small project adds another layer to my understanding.
